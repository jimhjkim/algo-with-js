# algo-with-js

Coding interview practice using JavaScript

## Useful commands

To install the JavaScript test framework **jest**...

```bash
npm install -g jest
```

To run a specific test file...

```bash
jest <exercise>/test.js --watch
```

## Debugging

node inspect <exercise>/index.js
